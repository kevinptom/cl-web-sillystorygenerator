A fun, interactive web application that generates random, silly stories at the click of a button! This project was built as part of the MDN Web Docs JavaScript First Steps module, enhanced with a modern, responsive user interface.

## 🚀 Live Demo

[Click here to view the Live Project!] https://kevinptom.github.io/cl-web-sillystorygenerator/

## ✨ Features

* **Custom Characters:** Enter any name to replace the default protagonist ("Bob") in the story.
* **Localization / Unit Conversion:** Toggle between US and UK radio buttons to dynamically convert temperatures (Fahrenheit to Centigrade) and weights (Pounds to Stones).
* **Dynamic Randomization:** Generates a completely new combination of events, locations, and actions every time you click the button.
* **Modern UI/UX:** Features a clean card layout, responsive design, interactive hover states, and a beautiful pastel gradient background.
